Product API Documentation

This API allows filtering, sorting, selecting fields, and pagination of products.

Endpoints

  • GET /api/v1/products - Get all products
  • GET /api/v1/products?search=chair - Search products by name/description
  • GET /api/v1/products?category=furniture - Filter by category
  • GET /api/v1/products?sort=price,-createdAt - Sort by price and createdAt (descending)
  • GET /api/v1/products?select=name,price - Select specific fields
  • GET /api/v1/products?page=2&limit=5 - Pagination

Numeric Filters

Filter products using numeric comparisons for fields like price and rating.

  • GET /api/v1/products?numericFilters=price>50 - Get products with price greater than 50
  • GET /api/v1/products?numericFilters=rating>=4 - Get products with rating 4 or higher
